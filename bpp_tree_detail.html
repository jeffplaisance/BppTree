template<typename Value, int LeafNodeBytes = 512, int InternalNodeBytes = 512, int depthLimit = 16, typename ...Ts>
class BppTreeDetail

A B++ tree is a B+ tree with a pluggable set of mixins

Template Parameters:

  • Value – The value type of the B++ Tree

  • LeafNodeBytes – The maximum number of bytes in a leaf node

  • InternalNodeBytes – The maximum number of bytes in an internal node

  • depthLimit – The maximum depth of the tree. Setting this lower reduces code size and compile times

  • Ts – The mixins to use with this B++ tree
